BrainNetAnno.neurotransmitter_pls_cge

PLS analysis for neurotransmitter contributions against FC deviations.

Provides reusable functions to select PLS components via CV, run PLS, compute permutation p-values for weights, and plotting utilities suitable for packaging.

Run PLS analysis linking FC weights to neurotransmitter contributions.
BrainNetAnno.neurotransmitter_pls_cge.run_neurotransmitter_pls_pipeline(fc_matrix_path: str, nt_contrib_path: str, top_k: int = 350, mse_plot_path: str | None = None, explained_variance_plot_path: str | None = None, output_weights_path: str | None = None, max_components: int = 10, cv_splits: int = 5, random_state: int = 42, n_permutations: int = 1000) → Tuple[int, pandas.DataFrame][source]

Run PLS analysis linking FC weights to neurotransmitter contributions.

This pipeline loads an FC weights matrix and a per-connection neurotransmitter contribution table, aligns by Region_Pair, selects the optimal number of PLS components via cross-validated MSE, fits the PLS model, computes permutation p-values for feature weights, saves results, and optionally saves plots of MSE and explained variance.

Parameters:

  • fc_matrix_path (str) – Path to the CSV containing a square FC weights matrix (no header).

  • nt_contrib (str) – Path to the CSV containing neurotransmitter contributions with a 'Region_Pair' column.

  • top_k (int, optional) – Number of top absolute FC entries to keep. Default 350.

  • mse_plot_path (Optional[str], optional) – Path to save the CV MSE curve plot. Default None.

  • explained_variance_plot_path (Optional[str], optional) – Path to save the explained-variance plot for X scores. Default None.

  • output_weights (Optional[str], optional) – Path to save weights and p-values CSV. Default None.

  • max_components (int, optional) – Maximum number of components considered in CV. Default 10.

  • cv_splits (int, optional) – Number of CV splits. Default 5.

  • random_state (int, optional) – Random seed for CV. Default 42.

  • n_permutations (int, optional) – Number of permutations for p-values. Default 1000.

Returns:

Best number of components and a DataFrame with neurotransmitter weights and p-values.

Return type:

Tuple[int, pd.DataFrame]

Notes

  • Output directories for plots and CSVs are created automatically when paths are provided.

  • The explained variance is computed from X scores relative to total X variance.
